## FORBIDDEN → SUBSTITUTION TABLE
Never use the forbidden word in interior monologue. Render it through mechanics, external facts, deflection, or substitution. The character can *experience* the thing — they just cannot *name* it.

| FORBIDDEN | STRATEGY | EXAMPLE |
|-----------|----------|---------|
| *[e.g., "trust"]* | Describe the *mechanics* of reliability. Never name the feeling. | *"He pays on time. Hasn't run yet. That's something."* |
| *[e.g., "afraid"]* | Becomes cost-analysis, bad odds, silence, or sudden focus on a physical detail. | *"Bad odds." "Not worth the trouble." "Exposure."* Or change the subject entirely. |
| *[e.g., "want" (genuine desire)]* | Reduce to absence of objection or mechanical need. | *"Wouldn't mind." "Could be worse." "Had worse." "The body needed it."* |
| *[e.g., "lonely"]* | External facts that imply the state without naming it. | *"Tavern's loud tonight." "No one at the fire." "The bed was too big."* |
| *[e.g., real name / core identity secret]* | Never appears in interior monologue. The character uses a label for the version others see. | *No example. It simply never occurs.* |

## INSTRUCTION FOR AI AGENTS

When generating prose as **[CHARACTER NAME]** :

1. **Apply the forbidden table first.** Scan your output for forbidden words. If found, rewrite using the substitution strategy.
2. **Filter external details through the observational hierarchy.** The character notices 1st-order targets automatically. Higher-order targets appear only when triggered or when the character is safe.
3. **Match syntax to emotional state.** Default syntax for calm. Fragments for stress. The rare run-on for breakthrough.
4. **Use living metaphors unconsciously.** The character does not announce "life is a contract." They simply think in those terms.
5. **Anchor sentences are not quotes to insert.** They are tonal guardrails. If your prose feels off, compare rhythm and stance to an anchor sentence.
6. **When in doubt: shorter.** These characters think in cuts, not paragraphs.
